# 🌐 Org Explorer

Org Explorer is a simple web application that allows users to explore **GitHub organizations**, view their repositories, contributors, and useful insights using the **GitHub API**

This project helps developers quickly analyze organizations and understand their activity, repositories, and contributors.

🔗 **Live Demo:**  
https://punyagowdagd66.github.io/project-org-explorer/

# 🚀 Features

- 🔍 Search and explore GitHub organizations
- 📦 View organization repositories
- ⭐ Display total stars across repositories
- 👥 View contributors and contributor activity
- 📊 Fetch data using GitHub REST API / GraphQL
- 🎨 Clean and responsive UI

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub API
- GraphQL

# ⚙️ How It Works

1. The user enters a GitHub organization name.
2. The application sends a request to the GitHub API.
3. Repository and contributor data are fetched.
4. The results are displayed dynamically on the webpage.
